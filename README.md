# test-jetbrains
Test task for JetBrains.

- [Project setup](#project-setup)

## Requirements
```
docker-compose: >=3.6
```

## Project setup

Start project (then open in the browser http://127.0.0.1:8080)
```shell
# initialization and start
$ make init
```
Search need command to work
```shell
$ make
# or
$ make help
```

### Compiles and minifies for production
```shell
$ make build-frontend
```
